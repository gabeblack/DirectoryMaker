DirectoryMaker by gabeblack

Easily creates a Ward Directory or a Ward Photo Directory. A stand-alone application that creates photo/telephone directories or rosters.

DirectoryMaker is an easy to use photo or text telephone directory maker. It was created to assist in creating an LDS Ward Photo Directory or a plain Ward Directory; capable of importing the CSV export from a Ward website or an MLS export of an LDS ward. However, it can also be used to create a photo directory for the use of any organization. Screenshots as well as a Windows MSI installer can be seen and downloaded from http://directorymaker.blogspot.com or ttp://warddirectory.blogspot.com

Dependencies

If running from source: * Python 2.5+ * WxPython * reportlab

Install

Windows easy: Download the MSI installer from http://directorymaker.blogspot.com All platforms: Install all dependencies (wxPython, reportlab) Download source Run "python DirectoryMaker.py"
